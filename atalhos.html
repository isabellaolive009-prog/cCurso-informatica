Atalhos do Teclado

Os atalhos de teclado ajudam você a trabalhar mais rápido e com mais eficiência. Eles permitem realizar ações sem precisar usar o mouse!

Copiar, colar e recortar

  • Ctrl + C → Copiar
  • Ctrl + V → Colar
  • Ctrl + X → Recortar
  • Ctrl + Z → Desfazer a última ação
  • Ctrl + Y → Refazer ação desfeita

Navegação e seleção

  • Ctrl + A → Selecionar tudo
  • Ctrl + F → Pesquisar na página
  • Alt + Tab → Alternar entre janelas abertas
  • Alt + F4 → Fechar janela atual

Atalhos do sistema

  • Windows + D → Mostrar área de trabalho
  • Windows + L → Bloquear o computador
  • Windows + E → Abrir o explorador de arquivos
  • Windows + Print Screen → Tirar print da tela

Outros úteis

  • Ctrl + Shift + Esc → Abrir o Gerenciador de Tarefas
  • PrtScn → Copiar imagem da tela
  • Alt + Enter → Tela cheia (em vídeos/jogos)
Dica: pratique esses atalhos no seu computador para memorizá-los com mais facilidade. Você pode abrir o Bloco de Notas e tentar copiar, colar e desfazer usando o teclado!
